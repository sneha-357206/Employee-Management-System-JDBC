# Employee Management System

## Technologies Used
- Java
- JDBC
- MySQL

## Features
- Insert Employee Record
- Update Employee Record
- Delete Employee Record
- Display Employee Records
- Add Column
- Delete Column

## Project Description
A menu-driven Employee Management System developed using Java, JDBC, and MySQL. The application performs CRUD operations and manages employee records stored in a MySQL database.

## Sample Menu

```text
1. Insert
2. Update
3. Delete
4. Display
5. Add Column
6. Delete Column
7. Exit
```
